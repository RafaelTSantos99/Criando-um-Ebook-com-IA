Este é um projeto acadêmico para criação de um podcast gerado por IA. Para que isto fosse possível foram usados diferentes prompts para geração de título, imagem e textos do Ebook. Seguem os prompts utilizados.

Geração do título:

  Crie um título de um ebook sobre o tema de css, o ebook é do nicho de programação e o subnicho é de css, o título deve ser épico e curto, e tenha uma temática de star wars no título, me liste 5 variações de títulos

  Geração do texto:

  Faça um texto para ebook , com foco em CSS, listando os principais seletores CSS com exemplos em código 
  
  {REGRAS} 
  Explique sempre de uma maneira simples 
  Deixe o texto enxuto 
  Sempre traga exemplos de código em contextos reais sempre deixe um título sugestivo por tópico
